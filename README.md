# mcp-trino-optimizer

A Model Context Protocol (MCP) server that helps LLM-driven clients (Claude Code, Claude Desktop, and other MCP-compatible tools) optimize Trino SQL queries running against Iceberg data lakes. It uses EXPLAIN / EXPLAIN ANALYZE evidence, a deterministic rule engine, and safe semantic-preserving rewrites to turn opaque query performance problems into actionable, evidence-backed reports.
